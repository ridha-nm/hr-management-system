# Interview Agent - Plan.md

**Status:** 🚧 Work in Progress
**Priority:** P0 (Tier 1 - Core HR)
**Created:** March 27, 2026

---

## 🎯 **Purpose**

Automate interview scheduling, coordination, and feedback collection between candidates and interviewers.

---

## 📋 **Features**

### **Interview Management**
- **Auto-Scheduling:** Match candidate & interviewer availability
- **Calendar Integration:** Google Calendar, Outlook
- **Video Links:** Auto-generate Zoom/Google Meet links
- **Room Booking:** Conference room allocation (for in-person)
- **Reminder System:** Email/SMS reminders (24h before)
- **Interview Kits:** Questions, resume, scorecard for interviewers
- **Feedback Collection:** Post-interview feedback forms
- **Rescheduling:** Handle schedule changes
- **Interview Types:** Phone, Technical, Manager, Cultural, Final

### **Commands**
- `schedule interview` - Start scheduling workflow
- `interview [id]` - Get interview details
- `reschedule interview` - Change interview time
- `interview questions [role]` - Generate questions
- `submit feedback` - Submit interviewer feedback
- `interview calendar` - View upcoming interviews
- `cancel interview` - Cancel scheduled interview
