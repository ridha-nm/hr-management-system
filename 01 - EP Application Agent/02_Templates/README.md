## 🚀 How to Use Templates

### Step 1: Start with Progress Tracker
```
1. Open: PROGRESS_TRACKER.md
2. Fill in candidate name
3. Review all 5 phases
4. Start with Phase 1
```

### Step 2: Get Documents
```
1. Open: DOCUMENT_CHECKLIST.md
2. Review required documents
3. Use EMAIL_TEMPLATES.md to request company docs
4. Collect candidate documents
```

### Step 3: MyFutureJobs
```
1. Open: MyFutureJobs_GUIDE.md
2. Follow step-by-step process
3. Post job (14 days minimum)
4. Conduct interviews
5. Use INTERVIEW_TEMPLATE.md to document
```

### Step 4: Continue to JTKSM & EP
```
1. Follow PROGRESS_TRACKER.md
2. Check off completed items
3. Use templates as needed
```
