## 📧 Email Templates

### Template: Request Documents from COO

```
Subject: Documents Required for EP Application - [CANDIDATE_NAME] ([POSITION])

Dear [COO Name],

We are proceeding with the EP application for:

Candidate: [CANDIDATE_NAME]
Position: [POSITION]

Required Documents:

1. Latest SSM Copy (printing date within 6 months)

2. Local Staff - Management Level (2 pax):
   - Signed Offer Letter / Employment Contract
   - Latest 3 months' payslips

3. Local Staff - General Worker (2 pax):
   - Signed Offer Letter / Employment Contract
   - Latest 3 months' payslips

4. All Directors in SSM:
   - IC or Passport detail page

5. Local Malaysia Staff Breakdown:
   - Top Management: ___ pax
   - Middle Management: ___ pax
   - Entry Level: ___ pax
   - Starting Basic Salary: RM ___

Please provide by [DATE].

Best regards,
[Your Name]
```

### Template: Request Job Description

```
Subject: Job Description Required - [POSITION]

Dear [Hiring Manager Name],

We require the finalized job description for:

Position: [POSITION]

Please include:
1. Key responsibilities
2. Reporting line
3. Required qualifications
4. Technical skills
5. Location

Deadline: [DATE]

Best regards,
[Your Name]
```

### Template: Request Tax Declaration from Candidate

```
Subject: EP Application - Additional Document Required

Dear [CANDIDATE_NAME],

We require the following document for your EP application:

Income Tax Declaration (IRB) - Previous employment in Malaysia

This is only required if you have previously worked in Malaysia.
Please provide your latest tax declaration slip.

Deadline: [DATE]

Best regards,
[Your Name]
```

## ⚠️ Important Notes

1. **SSM Copy:** Must be printed within 6 months of submission
2. **Payslips:** Latest 3 consecutive months
3. **Offer Letters:** Must be signed by both parties
4. **Directors' IC:** All directors listed in SSM
5. **CTC Stamps:** All educational certificates must have Certified True Copy stamp
6. **Tax Declaration:** Only if candidate worked in Malaysia before
7. **Passport Validity:** Minimum 15 months from intended travel date
