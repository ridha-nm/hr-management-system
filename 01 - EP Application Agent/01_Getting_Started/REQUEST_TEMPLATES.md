## 📧 Template 1: Request from COO (Email Format)

**To:** Syaheedah (COO)
**Email:** syaheedah@pantas.com
**Subject:** EP Application - Company Documents Required - [CANDIDATE_NAME]

```
Dear Syaheedah,

I hope this email finds you well.

We are proceeding with the Employment Pass (EP) application for:

Candidate: [CANDIDATE_NAME]
Position: [POSITION]
Nationality: [NATIONALITY]
Start Date: [DATE]

=== REQUIRED COMPANY DOCUMENTS ===

To complete the EP application, we require the following:

1. Latest SSM Copy
   - Printing date must be within 6 months
   - Required for: JTKSM + EP application

2. Directors' IC/Passport Details
   - For ALL directors listed in SSM
   - Format: Clear scan (front & back for IC)
   - Required for: Company verification

3. Employment Contracts + Payslips
   - Management Level: 2 pax + latest 3 months' payslips
   - General Worker: 2 pax + latest 3 months' payslips
   - Required for: JTKSM local staff verification
   - Note: Please send these directly to the immigration agency

4. Staff Breakdown Confirmation
   - Top Management: ___ pax
   - Middle Management: ___ pax
   - Entry Level: ___ pax
   - Starting Basic Salary: RM ___

=== TIMELINE ===

Target Submission Date: [DATE]
Documents Needed By: [DATE - 3 days]

=== NEXT STEPS ===

Once we receive these documents:
1. Complete candidate document collection
2. Submit MyFutureJobs application (14 days)
3. Generate JTKSM forms
4. Submit EP application

=== ATTACHMENTS ===

I've attached the complete document checklist for your reference.

Please let me know if you need any clarification or have questions about
these requirements.

Thank you for your support.

Best regards,
[YOUR NAME]
[YOUR POSITION]
[YOUR CONTACT]
```

## 📧 Template 3: Request from Candidate (Email Format)

**To:** Candidate
**Subject:** EP Application - Required Documents Checklist

```
Dear [CANDIDATE_NAME],

Congratulations on your selection for the [POSITION] position!

We are now proceeding with your Employment Pass (EP) application. To
complete the application, please provide the following documents:

=== REQUIRED DOCUMENTS ===

1. Passport Copy (ALL pages)
   ✓ All pages including blank pages
   ✓ Clear color scan
   ✓ PDF format

2. Passport-Sized Photo
   ✓ Light blue background
   ✓ Neutral expression (no smiling)
   ✓ Recent photo (within 3 months)
   ✓ White or light-colored clothing
   ✓ Size: 35mm x 50mm

3. CV/Resume
   ✓ Updated with latest experience
   ✓ Include all educational qualifications
   ✓ PDF format preferred

4. Educational Certificates
   ✓ All degree/diploma certificates
   ✓ MUST have "Certified True Copy" (CTC) stamp
   ✓ Clear color scans
   ✓ PDF format

5. Academic Transcripts
   ✓ Official transcripts from all institutions
   ✓ Clear color scans
   ✓ PDF format

6. IRB Tax Clearance (if applicable)
   ✓ Only if you previously worked in Malaysia
   ✓ Latest tax declaration slip (Form BE/B)

=== IMPORTANT REQUIREMENTS ===

⚠️ Passport Validity: Must be valid for at least 15 months from today
⚠️ CTC Stamps: Certificates without CTC stamp will be rejected
⚠️ Photo Background: Must be light blue (not white)

=== SUBMISSION ===

Please provide all documents by: [DATE]
Format: PDF via email
Subject line: "EP Documents - [YOUR NAME]"

=== NEXT STEPS ===

Once we receive your documents:
1. We'll verify all documents meet requirements
2. Submit MyFutureJobs application (14 days processing)
3. Generate and submit JTKSM forms
4. Submit EP application (4-6 weeks processing)

Total estimated processing time: 6-8 weeks

=== QUESTIONS? ===

If you have any questions or need clarification on any of these
requirements, please don't hesitate to contact me.

We look forward to receiving your documents soon.

Best regards,
[YOUR NAME]
[YOUR POSITION]
[YOUR CONTACT]
```
