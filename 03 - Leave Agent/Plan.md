# Leave Agent - Plan.md

**Status:** 🚧 Work in Progress
**Priority:** P0 (Tier 1 - Core HR)
**Created:** March 27, 2026

---

## 🎯 **Purpose**

Manage employee leave applications, approvals, balance tracking, and team leave calendar.

---

## 📋 **Features**

### **Leave Management**
- **Leave Types:** Annual, Sick, Emergency, Maternity, Paternity, Hajj, Unpaid
- **Leave Balance:** Real-time balance checker
- **Leave Application:** Submit leave requests
- **Approval Workflow:** Manager → HR (if required)
- **Leave Calendar:** Team leave visibility
- **Regional Holidays:** State-specific holiday calendars
- **Leave Encashment:** Calculate and process
- **Expiry Reminders:** Notify before leave expires

### **Commands**
- `apply leave` - Submit leave application
- `leave balance` - Check leave balance
- `leave history` - View leave history
- `team leave` - Team leave calendar
- `approve leave` - Manager approval
- `cancel leave` - Cancel approved leave
- `leave report` - Generate leave report

---

## 🔗 **Integrations**

- **Attendance Agent:** Absence tracking
- **Payroll Agent:** Unpaid leave deduction
- **Manager Dashboard:** Team leave visibility
